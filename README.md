# 🤔 How To Do

- 🧬 Clone This Repository
  ```bash
  git clone https://github.com/idgaf777/Blockmesh-.git
  ```
- 📂 Go to Folder
  ```bash
  cd Blockmesh
  ```
- ♦️ Install Dependencies
  ```bash
  bundle install
  ```
- 🏃🏻‍♂️‍➡️ Run the bot Script
  - if you want to use proxy setup first on proxy.txt
  - then run the script
    ```bash
    ruby main.rb
    ```
